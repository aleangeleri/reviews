# Ale & Sasi Reviews

Sito statico per raccogliere recensioni di ristoranti divise per localita.

## Avvio locale

```bash
node dev-server.js
```

Poi apri:

```text
http://127.0.0.1:5173
```

## Gestione contenuti

Il portale admin e disponibile da `#/admin`.

I dati vengono salvati nel `localStorage` del browser. Questo va bene per una prima versione locale/prototipo; per pubblicare il sito con login admin e dati condivisi servira collegare un backend o un CMS.
